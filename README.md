# `ssnshred`

With the proliferation of LLMs, many users may want to have an LLM check their tax forms.
Because tax forms contain SSNs, it is risky to simply upload a pdf to Anthropic/OpenAI/other LLM servers.
This repo contains a script for redacting SSNs from tax forms.
It has been tested on a tax return form generated by H&R Block.
